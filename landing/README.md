# Landing page (démo) — Chocolaterie du Bugnon

Cette landing page est une **démo frontend** (projet d’école) basée sur:
- `audit/chocolaterie-du-bugnon-audit.md`
- `audit/inspirations-ui-brainstorm.md`
- `audit/redesign-spec-bugnon.md`

## Ouvrir la page

Option la plus simple:
- Ouvrir `landing/index.html` dans ton navigateur.

Option “mini serveur” (recommandé si ton navigateur bloque des assets en local):

```bash
python3 -m http.server 5173
```

Puis ouvrir `http://localhost:5173/landing/`.

## Photos (d’où ça vient ?)
